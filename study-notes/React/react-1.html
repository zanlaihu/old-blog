1. 什么是React

React是用于构建用户界面的 JavaScript 库。具有渐进适配的特点，可以按需引入。

React 工具链

一个完整的 React 工具链包含：
1. package 管理器：yarn 或 npm
2. 打包器：webpack 或 parcel
3. 编译器：Babel，能让新版 JavaScript 在旧版浏览器运行

创建 React 工程

使用Create React App， 类似 Angular CLi，是一个脚手架工具。
Create React App 不会处理后端逻辑或操纵数据库，所以可以配合任何后端。其内部使用 Babel 和 webpack。


          npx create-react-app my-app

          cd my-app
          
          npm start

React 默认端口是 3000，与 Angular 的 4200 不同。
代码被修改后会自动重新编译，和 Angular 一样。但是 Angular 会自动刷新网站，React 则不会。（暂不明白为什么）
开发好后，执行 npm run build 会在 build 文件夹内生成应用的最优版本。

React元素

React由React元素构成。不管用JSX还是原生JavaScript编写都会创建对象，这些对象就是react元素。React DOM 通过读取React元素来更新 DOM 来与React元素保持一致： 


          // JSX
          const element = React.createElement(
            "h1",
            { className: "greeting" },
            "Hello, world!"
          );

          // 原生JavaScript
          const element = React.createElement(
            "h1",
            { className: "greeting" },
            "Hello, world!"
          );

          // 都是构建了这样一个对象（简化后）
          const element = {
            type: "h1",
            props: {
              className: "greeting",
              children: "Hello, world!",
            },
          };

根节点

React是单页面应用，从根节点开始渲染。


          <div id="root"></div>

类似Angular的：


          <app-root></app-root>

通过 ReactDOM.render()，将 React 元素渲染到根 DOM 节点：


          const element = <h1>Hello, world!</h1>;
          ReactDOM.render(element, document.getElementById("root"));
